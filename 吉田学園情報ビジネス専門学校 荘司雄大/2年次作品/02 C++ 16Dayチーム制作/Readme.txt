C++で6人で2週間で制作したゲームです。
自分の担当はモーション、ステージ制作、オブジェクト配置、テクスチャ、UIです。
他にも手が空いたら他の人の仕事を手助けにいくなど
積極的にチームに貢献するように動きました。

//-----------------------------------------------------------------------------
//  概要
//-----------------------------------------------------------------------------

●制作期間	        2週間
●プラットフォーム 	Windows10
●開発環境			VisualStudio2015 DirectX ver9.0
●使用言語			C++
●制作人数	 		6人
●ジャンル			アクション
●プレイ人数		１人


//-----------------------------------------------------------------------------
//  操作方法(キーボード)
//-----------------------------------------------------------------------------

●W/A/S/D/			移動
●SPACE				攻撃
●↑/↓/←/→		視点移動
●ENTER				ジャンプ/決定
●  P				ポーズ

//-----------------------------------------------------------------------------
//  操作方法(コントローラー)
//-----------------------------------------------------------------------------

●左スティック		移動
●Xボタン			攻撃
●Aボタン			ジャンプ/決定
●右スティック		視点移動
●スタートボタン	ポーズ
