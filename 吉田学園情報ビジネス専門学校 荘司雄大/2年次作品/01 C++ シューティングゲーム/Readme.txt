//-----------------------------------------------------------------------------

//  Absorber 〜アブソーバー〜
//
//-----------------------------------------------------------------------------

C++で初めて制作したゲームです。
1年生の時のシューティングゲームで貰った意見を反映させ、
より遊びやすいようなゲームを目指しました。

//-----------------------------------------------------------------------------
//  概要
//-----------------------------------------------------------------------------

●制作期間	        2か月
●プラットフォーム	Windows10
●開発環境			VisualStudio2015 DirectX ver9.0
●使用言語			C++
●制作人数 			1人
●ジャンル			アクションシューティングゲーム
●プレイ人数		1人

//-----------------------------------------------------------------------------
//  操作方法(キーボード)
//-----------------------------------------------------------------------------

●W/A/S/D/			移動
●SPACE				射撃
●  E				敵弾吸収
●ENTER				決定
●  P				ポーズ
*ゲージ使用スキル*
●	X				ヒール(ゲージ半分)
●	Q				必殺技(ゲージ全消費)
●↑/↓/←/→		必殺技操作

*デバッグ用操作*
●  Z				自機体力減少
●  G				スキルゲージ増加

//-----------------------------------------------------------------------------
//  操作方法(DirectInputコントローラー)
//-----------------------------------------------------------------------------

●左スティック/十字キー			移動
●Rボタン						射撃
●Lボタン						敵弾吸収
●スタートボタン				ポーズ

*ゲージ使用スキル*
●左スティック押し込み			ヒール(ゲージ半分)
●右スティック押し込み			必殺技(ゲージ全消費)
●右スティック					必殺技操作
