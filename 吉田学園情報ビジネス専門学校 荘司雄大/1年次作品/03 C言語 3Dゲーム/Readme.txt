//-----------------------------------------------------------------------------
//
//  Spin a Top
//
//-----------------------------------------------------------------------------

対戦型3Dアクションゲームです。
1人モードはNPCとの対戦で、
2人モードは矢印キーとテンキーを使うことで対戦が出来ます。

//-----------------------------------------------------------------------------
//  概要
//-----------------------------------------------------------------------------

●制作期間		        2か月
●プラットフォーム	 	Windows10
●開発環境				VisualStudio2015 DirectX ver9.0
●使用言語				C言語
●制作人数		 		1人
●ジャンル				アクション
●プレイ人数			1〜2人

//-----------------------------------------------------------------------------
//  操作方法
//-----------------------------------------------------------------------------
　*基 本 操 作*
●ENTER  		モードセレクト展開
●SPACE	   		決  定
●	P			ポーズ

　*プレイヤー１*
●W/A/S/D  		移 動
●SPACE	   		射 撃

　*プレイヤー２*
●↑/↓/←/→  	移 動
●NumEnter	   	射 撃
